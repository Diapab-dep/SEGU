1. Resumen Ejecutivo

Este informe presenta una comparacion objetiva entre el costo y tiempo real invertido en el desarrollo del Asistente de Envios Internacionales mediante interaccion con Inteligencia Artificial — sin conocimientos previos de desarrollo de software por parte del ejecutor — versus el costo equivalente bajo la metodologia tradicional de Fabrica de Software, tomando como referencia la propuesta comercial de Tata Consultancy Services (TCS) — Linea Base & Ondemand — Enero 2025.

Conclusion principal: El proyecto fue ejecutado en 3 semanas con un costo total de $947.636 COP (herramienta IA + tiempo interno del Director IT valorado a tarifa de nomina), logrando un ahorro de $31.821.820 COP respecto a la opcion TCS Ondemand. Esto representa un ROI del 3.358% en escenario conservador, o 38.911% considerando unicamente el costo marginal de la herramienta ($84.000 COP).

2. Alcance del Proyecto Construido

El sistema entregado incluye los siguientes componentes, todos funcionando en produccion en Azure VM (IP: 20.102.51.145) a la fecha de este informe:

2.1. Aplicacion Web y Logica de Negocio
  • Portal web con autenticacion individual por usuario (Flask / Python)
  • Dos roles diferenciados: Administrador y Operador, con menus adaptados
  • Interfaz de consulta con IA integrada — Anthropic Claude Haiku 4.5 (migrado desde Groq el 09-Mar-2026)
  • Deteccion automatica del pais en la pregunta y filtrado de contexto (22 paises, aliases en espanol e ingles)
  • Sistema de reglas directas (respuestas sin consumo de API)
  • Bloqueo de prompt injection en 2 capas: regex frontend + system prompt reforzado
2.2. Panel Administrativo y Gestion de Datos
  • Gestion completa de usuarios: crear, editar, activar/desactivar, eliminar
  • Actualizacion de la Matriz de Consulta (.xlsb) con previsualizacion de cambios
  • Comparacion automatica de matrices (nueva vs. anterior): paises, categorias, celdas modificadas
  • Backup automatico del archivo fuente (.xlsb.bak) antes de aplicar cambios
2.3. Infraestructura y Despliegue
  • Dockerfile non-root (appuser UID 1001) con health check cada 30 segundos
  • Imagen Docker desplegada en Azure VM Ubuntu 24.04 — IP publica: 20.102.51.145
  • Nginx como reverse proxy con HTTPS (TLS 1.2/1.3 — autofirmado) y redireccion HTTP→HTTPS
  • Red Docker aislada (inter-net) para comunicacion entre contenedores
  • Pipeline CI/CD con GitHub Actions (self-hosted runner) — despliegue automatico en push a main
  • Reglas NSG Azure: puertos 22, 80, 443, 5000 configurados

6. Analisis de Retorno sobre la Inversion (ROI)
6.1. Escenario A — Costo Conservador (incluye tiempo interno valorado)
Inversion real = Tiempo Director IT + Herramienta IA = $863.636 COP + $84.000 COP = $947.636 COP Ahorro neto = Costo TCS Ondemand - Inversion real = $32.769.456 COP - $947.636 COP = $31.821.820 COP ROI = ( Ahorro / Inversion ) x 100 = ( $31.821.820 / $947.636 ) x 100 = 3.358 %

Por cada $1 COP de costo total (incluyendo el tiempo del Director IT), se generaron $34,58 COP de valor equivalente.

6.2. Escenario B — Costo Marginal (gasto adicional real para la empresa)
Inversion marginal = Solo herramienta IA = $84.000 COP (El salario del Director IT es costo fijo, independiente del proyecto) Ahorro neto = $32.769.456 COP - $84.000 COP = $32.685.456 COP ROI = ( $32.685.456 / $84.000 ) x 100 = 38.911 %

Por cada $1 COP de gasto adicional para la empresa, se generaron $390 COP de valor.

7. Capacidades Habilitadas con IA no Contempladas por TCS

La propuesta TCS esta orientada a mantenimiento post-migracion de productos existentes (Hipolita, Flybox, Socrates, Deprisa.com). Las capacidades especificas de IA generativa para consultas de negocio no estan contempladas en ningun rol de la propuesta TCS, lo que implicaria contratar adicionalmente:

Rol adicional requerido Tarifa/hora TCS Motivo
Data Scientist ML – IA $121.629 Integracion y afinamiento del modelo LLM
Arquitecto especialista Azure $158.942 Despliegue en contenedores Docker + Azure
Experto DevOps $123.879 CI/CD con GitHub Actions, nginx, SSL
Costo hora combinado (soporte recurrente) $404.450 / hora Si se requiriera soporte mensual en estos roles
8. Recomendaciones para Gerencia

1. Escalar el modelo IA-asistida:
El patron aplicado en este proyecto es replicable en otras automatizaciones del area IT (integraciones, reportes, Hipolita, Flybox). Cada proyecto replicado multiplica el ROI acumulado.

2. Presupuesto IA sugerido:
Una suscripcion Claude Code Pro ($20 USD/mes) mas Anthropic API (~$20 USD/mes) representa menos del 0.2% del costo mensual TCS Linea Base ($66.4M COP/mes), con capacidad de ejecutar multiples proyectos simultaneos.

3. ✅ Migracion a Anthropic Claude — COMPLETADO (09-Mar-2026):
El LLM de produccion fue migrado de Groq a Anthropic Claude Haiku 4.5. La precision de las respuestas mejoro significativamente. Costo: ~$10–15 USD/mes.

9. Conclusion

El proyecto Asistente de Envios Internacionales demuestra que es posible desarrollar, asegurar y desplegar en produccion un sistema de software funcional, seguro y completamente documentado en 3 semanas y con una inversion de $947.636 COP (escenario conservador), utilizando Inteligencia Artificial como asistente de desarrollo, sin requerir un equipo de tecnologia dedicado ni conocimientos previos de programacion por parte del ejecutor.

El ahorro generado en este unico proyecto ($31.821.820 COP) equivale a:
  • Financiar 32 anos de suscripcion Claude Code Pro ($84.000 COP/mes)
  • O el 1,7% del costo anual TCS Linea Base ($66.4M/mes × 12 = $797M COP/ano)
  • O 18 dias de nomina completa del equipo TCS (4 personas, $66.4M/mes)
  • O 11 proyectos adicionales de complejidad similar al costo marginal de la herramienta IA

Esto posiciona la metodologia IA-asistida como estrategicamente transformadora para Latin Logistics / Deprisa, no solo como una alternativa de ahorro puntual, sino como un nuevo modelo de capacidad interna para la Gerencia Digital y Tecnologia.
